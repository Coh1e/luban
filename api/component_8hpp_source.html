2// 1:1 port of luban_boot/component.py 的 install/uninstall pipeline。
3//
4// install pipeline:
5// 1. manifest_source::load(name) — overlay or in-tree seed only, no network
6// 2. parse via scoop_manifest（含安全白名单）
7// 3. 算 toolchain_dir 名 = "<name>-<version>-<arch>"
8// 4. 幂等：installed.json 已记录同版本且目录存在 → 直接返回
9// 5. download archive 到 cache（带 SHA 校验，缓存命中也再校验一次）
10// 6. extract 到 staging dir（toolchains/.tmp-...）
11// 7. apply extract_dir：定位 toolchain 真根
12// 8. promote staging → final（os::rename，跨卷 fallback copy）
13// 9. 给每个 bin 写 shim
14// 10. 写 installed.json record（atomic）
15//
